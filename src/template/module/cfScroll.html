簡介
捲軸自動滑動到 DOM 定點的功能
Depend onplugin scrollTo of gsap
參數
$('xxx').cfScroll($contentArray,[opt = {}]);
$contentArray: $self 裡面的 dom Array
opt 參數
basicY: 判定捲動到定點的 Y 軸基準線, 可以是回傳 num 的 fn, 預設為本身高度的一半
offsetTop: 每個 $content 的 top 基準位移
isLiveTop: 是否要及時更新 top 資訊
duration: 捲動秒數, 設為 0 代表直接跳到該位置, 預設 1
ease: gsap 捲動 ease 效果, 預設 Power2.easeOut
befScroll: 捲動前的 cb(nowIndex, event), return false 則不捲動
onScroll: 捲動時的 cb(nowIndex, event), return false 則不捲動
onScroll: 捲動時的 cb(nowIndex, event)
aftScroll: 捲動後的 cb(nowIndex, event)
函式
getNowIndex(): 取得現在所在的 $content index
scrollNext([duration]): 捲動到下一個
參數
duration: 捲動秒數
scrollPrev([duration]): 捲動到上一個
參數
duration: 捲動秒數
scrollPrev(i, [duration]): 捲動到指定的 index
參數
i: 在 $content arr 裡的 index
duration: 捲動秒數
updateTop(): 重新掃描內容的高度位置
bindScroll(): 綁定 scroll
unbindScroll(): 不綁定 scroll
