簡介
表單驗證, 要檢查的欄位需要設置 id 或 name
參數
$('xxx').cfValidator([opt = {}]);
opt 參數
requireAll: 是否全部必填, 預設 false
函式
以下的參數 $dom 都需要有 id 或是 name, 用來記錄驗證項目
setRequire($dom): 設置檢查必填
setNum($dom): 設置檢查數字
setMinLength($dom, length): 設置檢查最小長度
setWord($dom): 設置檢查只能英數字
setEmail($dom): 設置檢查 email 格式
setMobile($dom): 設置檢查手機格式
setRocId($dom): 設置檢查身分證格式
validate(): 開始檢查格式程式範例
html
<form id='form1'>
<input type='text' id='address' />
<input type='text' id='phone' />
</form>
js
var validator = $('#form1').cfValidator({requireAll: true});
// 全部都有填寫 => errInfo = null;
// 全部都沒填寫 => errInfo = {address: ['require'], phone: ['require']}
var errInfo = validator.validate();
