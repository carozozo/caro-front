cfModal
示範
開啟 modal 視窗
This is Modal
簡介
modal 視窗
參數
$('xxx').cfModal([opt = {}])
opt 參數
isClickClose: 是否點選內容之外的部分就 close modal, 預設 true
befShow: 顯示視窗之前觸發的 cb(), return false 則不顯示
aftShow: 顯示視窗之後觸發的 cb()
befHide: 關閉視窗之前觸發的 cb(), return false 則不關閉
aftHide: 關閉視窗之後觸發的 cb()
函式
showModal(): 顯示視窗
hideModal(): 關閉視窗
