cfRandomDrop
示範
預設
請在此範圍移動或點擊滑鼠
簡介
隨機產生滑落物件, 例如氣泡, 水滴效果
參數
$('xxx').cfRandomDrop($imgArr, [opt = {}])
$imgArr: DOM 陣列
minStartX: 最小 x 起點, 可以是回傳 int 的 fn, 預設 0
maxStartX: 最大 x 起點, 可以是回傳 int 的 fn, 預設取得本身寬度
minStartY: 最小 y 起點, 可以是回傳 int 的 fn, 預設 0
maxStartY: 最大 y 起點, 可以是回傳 int 的 fn, 預設取得本身高度
minRandomMs: 間隔最短多少毫秒產生新滑落物件, 預設 0
maxRandomMs: 間隔最長多少毫秒產生新滑落物件, 預設 minRandomMs + 1000
inStartDuration: 滑落物件產生時, 要停留在原地的秒數, 預設 0
inEndDuration: 滑落物件移動完成時, 要停留在原地的秒數, 預設 0
minDuration: 最小移動秒數, 預設 3
maxDuration: 最大移動秒數, 預設 5
minDistance: 最小移動距離, 預設 0
maxDistance: 最大移動距離, 預設取得本身高度
minScale: 每次隨機產生的滑落物件, 最小縮放值, 預設 1(不縮放)
rotationRange: 移動時的旋轉角度範圍, 預設 0
xRange: 橫向移動範圍, 預設 0
reverse: 是否反轉 drop 方向
amount: 每次產生的物件數量, 預設 1
函式
startDrop(): 開始產生滑落物件
stopDrop(): 停止產生滑落物件
clickCreate([nameSpace]): 當 click 時, 產生滑落物件
參數
nameSpace: on click 的 name space
moveCreate([pinterval = 10], [nameSpace]): 當 mousemove, 產生滑落物件
參數
interval: mousemove 觸發幾次後產生滑落物件, 數字越接近 0 越快
nameSpace: on mousemove 的 name space
