<div id='content'>

html
var cir = $('#ccontent').cfCircleShow([$('#pic1'), $('#pic2')]);
cir.autoPlay();
...
cir.stopPlay();
$targetList: 要輪播的 $dom Array
opt 參數
zIndex: 當前 $target 的 z-index, 預設 999
radios: 旋轉半徑, 預設 100
duration: 移動時間, 預設 1
minBrightness: 最後面的 DOM 的亮度百分比, 預設 50
degreeTop: 每個 $dom 的 top 位移量, 預設 0
minScale: 最後面的 $target 的 scale, 預設 1
minScale: TweenMax ease 效果
cb: 每次移動的 cb($dom, i), 可取得目前最前面的 $dom 和 index
函式
next(): 顯示下一個內容
prev(): 顯示上一個內容
getCurrentIndex(): 取得目前內容的 index
autoPlay(ms, [isNext = true]): 自動輪播
參數
ms: 每次移動的毫秒數, 預設 1000
isNext: 輪播方向, true 為順時針
stopPlay(): 停止輪播
