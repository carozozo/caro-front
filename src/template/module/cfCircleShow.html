cfCircleShow
簡介
DOM circle 輪播效果
參數
$targetList: 要輪播的 DOM Array opt: options
opt 參數
zIndex: 當前 $target 的 z-index, 預設 999
radios: 旋轉半徑, 預設 100
duration: 移動時間, 預設 1
minBrightness: 最後面的 DOM 的亮度百分比, 預設 50
degreeTop: 每個 DOM 的 top 位移量, 預設 0
minScale: 最後面的 $target 的 scale, 預設 1
minScale: TweenMax ease 效果
cb: 每次移動的 cb($target, i), 可取得每個 DOM 及該 DOM 目前的 index
函式
next(): 顯示下一個內容 範例
var cir = $('xxx').cfCircleShow([...]);
cir.next();
prev(): 顯示上一個內容 範例
var cir = $('xxx').cfCircleShow([...]);
cir.prev();
getCurrentIndex(): 取得目前內容的 index 範例
var cir = $('xxx').cfCircleShow([...]);
var index = cir.getCurrentIndex();
autoPlay(ms, [isNext = true]): 自動輪播 範例
參數
ms: 每次移動的毫秒數, 預設 1000
isNext: 輪播方向, true 為順時針
var cir = $('xxx').cfCircleShow([...]);
cir.autoPlay();
stopPlay(): 停止輪播 範例
var cir = $('xxx').cfCircleShow([...]);
cir.autoPlay();
...
cir.stopPlay();
