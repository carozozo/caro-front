cfPiece
示範
y 切成 等份,x 切成 等份
分割
滑鼠移到圖片上可看到切片的順序
簡介
切片效果, 切片後 $self 會隱藏
注意切片後為固定寬高, 只繼承 $self 的 css position
切片後請自行調整 css 以避免跑版
參數
$('xxx').cfPiece([particleY = 3], [particleX = 3]);程式範例
html
<div id='img' style='position:static;'>...</div>
js
var $img = $('#img').cfPiece(2, 3);
// $piece1~$piece6 即產生的 jQuery 切片物件
var $pieceArr = $img.$pieceArr; // [$piece1, $piece2, $piece3, $piece4, $piece5, $piece6]
// $piece1~$piece6 即產生的 jQuery 切片物件
var $pieceTable = $img.$pieceTable; // [ [$piece1, $piece2, $piece3], [$piece4, $piece5, $piece6] ]
每個切片被賦予的 _yIndex 和 _xIndex
var index1 = $piece1._yIndex + '-' + $piece1._xIndex; // '0-0
var index2 = $piece2._yIndex + '-' + $piece1._xIndex; // '0-1'
...
var index4 = $piece4._yIndex + '-' + $piece1._xIndex; // '1-0'
產生的切片 html 結構
<!-- $self 被設為隱藏 -->
<div id='img' style='position:static; display:none;'>...</div>
<!-- $img.$pieceContainer -->
<div class='cfPieceContainer' style='position: static;'>
<!-- $img.$pieceInnerContainer -->
<div class='cfPieceInnerContainer' style='position: relative;'>
<!-- 每個產生的切片 -->
<div class='cfPiece'>...</div>
</div>
</div>
particleY: y 切成幾等份
particleX: x 切成幾等份
$self 附加屬性
$pieceArr: 儲存每個切片
$pieceTable: 儲存切片陣列, 例如切成 2 x 3, 陣列就是 [[$piece1, $piece2, $piece3],[$piece4, $piece5, $piece6]]
$pieceContainer: 外部切片的容器, 繼承 $self 的 css position 屬性
$pieceInnerContainer: 內部切片容器, 協助切片定位
$piece 附加屬性
_yIndex: 儲存切片在 y 軸的 index
_xIndex: 儲存切片在 x 軸的 index
