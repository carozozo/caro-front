簡介
計算滑鼠和基準點的距離，同步移動 DOM
參數
$('xxx').cfSyncMove(nameSpace, [opt = {}])
nameSpace: 綁定 mousemove 和 touchmove 的 name space
opt 參數
baseX: x 基準坐標, 不是數字或 return 數字的 fn 則不同步移動 x, 預設為視窗寬度一半
baseY: y 基準坐標, 不是數字或 return 數字的 fn 則不同步移動 y, 預設為視窗高度一半
proportionX: x 隨者滑鼠的移動比例, 預設 0.05
proportionY: y 隨者滑鼠的移動比例, 預設 0.05
rangeX: x 最大移動範圍, 未設置代表不限制
rangeY: y 最大移動範圍, 未設置代表不限制
reverseX: x 移動方向是否和滑鼠方向相反, 預設 true
reverseY: y 移動方向是否和滑鼠方向相反, 預設 true
befMove: 移動之前的 cb(infoObj), 如果 return false 則不移動
infoObj 項目
event: event
moveX: x 移動距離
moveY: y 移動距離
mouseX: 滑鼠 x 座標
mouseY: 滑鼠 y 座標
baseX: 取得的 x 基準坐標
baseY: 取得的 y 基準坐標
$self: Dom 本身, e.g. $('xxx').cfSyncMove() => $self = $('xxx')
aftMove: 移動之後的 cb(infoObj)
infoObj 項目
event: event
moveX: x 移動距離
moveY: y 移動距離
mouseX: 滑鼠 x 座標
mouseY: 滑鼠 y 座標
baseX: 取得的 x 基準坐標
baseY: 取得的 y 基準坐標
$self: Dom 本身, e.g. $('xxx').cfSyncMove() => $self = $('xxx')
stopMove: 停止移動之後的 cb
函式
startSyncMove: 開始監聽滑鼠並同步移動
stopSyncMove: 停止監聽滑鼠
