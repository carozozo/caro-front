cf.router
簡介
換頁函式庫, 使用 $.ajax 切換頁面, 並利用 url hash 記錄
設定檔
container: 頁面容器的 html id, 沒設置時, 預設容器為 $('body')
屬性
$container : 當前頁面的容器 DOM 
$page : 當前頁面 DOM
_befPage : 紀錄換頁前會執行的 fn s
_prePage : 紀錄載入頁面前會執行的 fns 
_page : 當載入頁面後要執行的指定 fn  
_aftPage : 紀錄載入頁面後會執行的 fn s
_transitionFn : 紀錄當前換頁效果 fn 
pageName : 當前頁面名稱 
函式
getPageByHash([hash]) : 依照 url 的 hash 判斷目前的 page
Get Page Name
getSearchByHash([hash]) : 依照 url 的 hash 判斷目前的 search
Get Search
parseUrlSearch() : 依照 url 的 hash 裡面的 search 轉為 obj
Get SearchObj
regBefPage(fn, [執行順序=50]) : 註冊每次換頁前要執行的 fn
regPrePage(fn, [執行順序=50]) : 註冊每次載入頁前要執行的 fn
regAftPage(fn, [執行順序=50]) : 註冊每次載入頁後要執行的 fn
regPage(註冊頁面, fn) : 註冊載入指定頁面後要執行的 fn
goPage([頁面名稱]) : 換頁, 沒參數時會依據 url 的 hash 自動判斷
goPage('router')
goPage()
goPage('router?name=pikachu&age=200')
blockGoPage() : 呼叫後, 執行 router.goPage 不會換頁
Block Go Page
approveGoPage() : 呼叫後, 所有的 router.goPage 可以換頁  
Approve Go Page
goPage 執行流程
執行儲存在 _befPage 裡的 fns
取得要跳轉的頁面名稱
透過 ajax 從 template 取得對應的 html
執行儲存在 _prePage 裡的 fns
移除原本的 page 內容, 並置入目標頁面內容
執行對應的 _page fn
執行儲存在 _aftPage 裡的 fns
