cf.fb
簡介
FB API 延伸函式庫, 解決部分 API 問題, 簡化呼叫方式和防呆 官方文件
設定檔
isDownloadSdk: 是否要載入 fb-sdk
sdkVersion: fb-sdk 版本, e.g. 'v2.7'
redirectAfterLogin: 當登入 fb 後, 要跳轉的 template 頁面, 沒設時會回當下首頁
appId: 網站所使用的 app-id
shareUrl: 分享網址, 需要可以在網路上找的到的 url, 所以不能用 localhost
函式
regInitCb(name, cb): 註冊當 fb-sdk init 完成後要執行的 cb
isReady(): 判斷 FB 是否 ready
getAppId(): 取得 FB APP id
getUserId(): 取得 user id
getAccessToken(): 取得 access token
getLoginStatus(): 取得登入狀態, 同FB.getLoginStatus, 另支援函式 .suc 和 .err; 官方文件
範例
cf.fb.getLoginStatus().suc(function(){
  // 狀態是登入
}).err(function(err){
  // 狀態是未登入
});
login([opt={}]): 登入, 同 FB.login, 另支援函式 .suc 和 .err; 官方文件
opt 參數
同 FB.login 的 options
範例
// 會自動開啟 fb login 視窗
cf.fb.login().suc(function(){
  // 狀態是登入
}).err(function(err){
  // 狀態是未登入
});
logout(): 登出, 同 FB.logout, 另支援函式 .suc 和 .err; 官方文件
範例
cf.fb.logout().suc(function(){
  // 狀態是登入
}).err(function(err){
  // 狀態是未登入
});
getProfile(fieldArr=[]): 取得 user 資料, 另支援函式 .suc 和 .err 官方文件
fieldArr
資料欄位陣列, 預設['id', 'name', 'email', 'gender']
範例
cf.fb.getProfile().suc(function(res){
  // 成功取得 res
}).err(function(err){
  // 呼叫失敗
});
feed([opt={}]): 分享 fb-app, 另支援函式 .suc 和 .err; 官方文件
opt 參數
同官方文件; 分享連結由 config 設定, 或自動取得目前所在的 url
範例: 設定檔 shareUrl = 'https://carozozo.github.io/caro-front/'
// 會自動開啟 fb feed 視窗
cf.fb.feed().suc(function(res){
  // 分享成功
}).err(function(err){
  // 分享失敗
});
