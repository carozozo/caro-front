程式範例
var $page = $('.cf-page');
var fileArr = ['blacktocat.png', 'caro-front.png', 'cf_title1.png', 'cf_title2.png', 'cf_title3.png', 'no_such_file.png', 'cf_title4.png', 'cf_title5.png', 'hand.png'];
fileArr = caro.map(fileArr, function(file) {
return 'images/' + file;
});
$percentage = $page.find('#percentage');
$page.cfPreLoad(fileArr, {
onLoadStart: function(e) {...},
onComplete: function(e) {...},
onError: function(e) {...},
onProgress: function(e, progress) {
var percentage = Math.round(progress * 100);
$percentage.html(percentage);
}
});
$page.find('#startLoadBtn').on('click', function() {
$page.startLoad();
return;
});
簡介
預載檔案
depend onpreloadjs-0.4.1
參數
$('xxx').cfPreLoad(fileArr, [opt={}])
fileArr: 檔案路徑陣列
opt 參數
onLoadStart: 開始讀取時觸發的 cb(event)
onComplete: 全部讀取完成時觸發的 cb(event)
onError: 發生錯誤時觸發的 cb(event)
onProgress: 進行讀取時觸發的 cb(event, progress); progress 為 0~1 讀取進度
函式
startLoad: 開始執行載入
