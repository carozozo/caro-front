cf.routeAnimate
簡介
擴充換頁效果, 換頁功能請先參考 cf.router
Depend ongsap
函式
left([opt = {}]): 頁面左移換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
right([opt = {}]): 頁面右移換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
scale([opt = {}]): 縮放換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
rotation: 旋轉角度, 預設 0
fade([opt = {}]): fade 換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
slide([opt = {}]): slide 換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
piece([opt = {}]): 切片換場效果
opt 參數
duration: 換頁時間, 預設 0.8 秒
pieceX: x 要切成幾等份
pieceY: y 要切成幾等份
clear([opt = {}]): 清除換場效果
