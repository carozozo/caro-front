cf.unit
簡介
一些單元程式
設定檔
imgPath: 放置圖片的路徑, 預設 images
函式
open(URL, [name], [specs], [replace]): window.open 進階版, 偵測被瀏覽器封鎖彈出視窗時會 alert 訊息
ifJquery(arg): 判斷是否為 jQuery 物件
getImgSize($img, cb): 取得圖片真實寬高程式範例
假設 html 裡有圖片 <img id="img" src="images/logo.png"/>
cf.fb.getImgSize($('#img'), function(width, height){
...
});
getImgPath([fileName='']): 取得圖片路徑程式範例
設定檔 imgPath = 'https://carozozo.github.io/caro-front/images'
cf.unit.getImgPath('logo.png');  // 'https://carozozo.github.io/caro-front/images/logo.png'
replaceImgPath($dom, [imgPath = config.imgPath]): 轉換 $dom 的圖片路徑, 由 css 設定的同樣有效程式範例
假設 html 裡有圖片 <img id="img" src="images/logo.png"/>
// <img id="img" src="https://carozozo.github.io/caro-front/images/logo.png"/>
cf.unit.replaceImgPath($('#img'),'https://carozozo.github.io/caro-front/images');
假設 html 裡有 <div id="bg" style="background:url(images/bg.jpg)"/>
// <img id="img" style="background:url(https://carozozo.github.io/caro-front/images/bg.jpg)"/>
cf.unit.replaceImgPath($('#bg'),'https://carozozo.github.io/caro-front/images');
