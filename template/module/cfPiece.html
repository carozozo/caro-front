cfPiece
示範
y 切成 等份,x 切成 等份
分割
回復
滑鼠移到圖片上可看到切片的順序
簡介
切片效果, 切片後 $self 會隱藏
注意切片後為固定寬高, 只繼承 $self 的 css position
切片後請自行調整 css 以避免跑版
參數
$('xxx').cfPiece([particleY = 3], [particleX = 3], [opt = {}]);程式範例
html
<div id='img' style='position:static;'>...</div>
js
var $img = $('#img').cfPiece(2, 3, {
befPiece: function(yIndex, xIndex) {
...
},
aftPiece: function($piece, yIndex, xIndex) {
...
}
});
產生的切片 html 結構
<!-- $self 被設為隱藏 -->
<div id='img' style='position:static; display:none;'>...</div>
<!-- $img.$pieceContainer -->
<div class='cfPieceContainer' style='position: static;'>
<!-- $img.$pieceInnerContainer -->
<div class='cfPieceInnerContainer' style='position: relative;'>
<!-- 每個產生的 $piece 切片 -->
<div class='cfPiece'>...</div>
</div>
</div>
particleY: y 切成幾等份
particleX: x 切成幾等份
opt 參數
pefPiece: 每次置入 $piece 前呼叫的 cb(yIndex, xIndex), return false 則不值置入
aftPiece: 每次置入 $piece 後呼叫的 cb($piece, yIndex, xIndex)
$self 附加屬性
$pieceContainer: 外部切片的容器, 繼承 $self 的 css position 屬性
$pieceInnerContainer: 內部切片容器, 協助切片定位
函式
reversePiece: 回復成切割前的狀態
