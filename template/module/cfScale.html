cfScale
示範
拖拉視窗觀看變化
簡介
隨目標縮放自己的大小
參數
$('xxx').cfScale(nameSpace, [opt={}])
nameSpace: 綁定 $target resize 時的 name space
opt 參數
$target: 要以哪個目標的大小為依據做 scale, 預設 $(window)
startX: 當 $target 寬介於 startX 和 endX 時, 觸發 scale, 預設 1280
endX: 當 $target 寬介於 startX 和 endX 時, 預設 1920
startY: 當 $target 高介於 startY 和 endY 時, 觸發 scale, 預設 720
endY: 當 $target 高介於 startY 和 endY 時, 觸發 scale, 預設 1080
startScale: 最小 scale 值, 預設 1
endScale: 最大 scale 值, 預設 1.2
mode: scale 基準, 'min'=> scaleX scaleY 取最小, 'max'=> scaleX scaleY 取最大, 'x'=> 取 scaleX, 'y'=> 取 scaleY, 預設 'x'
isScaleX: 是否縮放寬度, 預設 true
isScaleY: 是否縮放高度, 預設 false
duration: 每次執行 scale 的秒數, 預設 0.3
befScale: scale 之前的 cb(infoObj), 回傳 false 會停止 scale
aftScale: scale 之後的 cb(infoObj)
basicWidth: 設置本身起始的 width, 預設取 $self.width()
basicHeight: 設置本身起始的 height, 預設取 $self.height()
函式
updateScale(): 計算並執行 scale
bindScale(): 綁定 scale
unbindScale(): 不綁定 scale
setBasicWidth(width): 設定 scale width 起始值
setBasicHeight(height): 設定 scale height 起始值
