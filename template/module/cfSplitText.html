簡介
將目標內的文字拆解成 div
參數
$('xxx').cfSplitText([opt ={}]);程式範例
html
<div id='demoText'>Caro Front</div>
js
var $demoText = $('#demoText').cfSplitText({
isToChar: true,
isToWord: true,
charCb: function($char) {
...
},
wordCb: function($word) {
...
}
});
$demoText.reverseText();
產生的切片 html 結構
<div id='demoText'>
<div class='cfSplitTextWord'>
<div class='cfSplitTextChar'>C</div>
<div class='cfSplitTextChar'>a</div>
<div class='cfSplitTextChar'>r</div>
<div class='cfSplitTextChar'>o</div>
<div class='cfSplitTextWord'>
<div class='cfSplitTextChar'>F</div>
<div class='cfSplitTextChar'>r</div>
<div class='cfSplitTextChar'>o</div>
<div class='cfSplitTextChar'>n</div>
<div class='cfSplitTextChar'>t</div>
</div>
</div>
opt 參數
isToChar: 是否拆解每個字元, 預設 true
isToWord: 是否以空白字元拆解, 拆解後稱為字組, 預設 false
charCb: 放置每個字元之後呼叫的 cb($char, i)
wordCb: 放置每個字組之後呼叫的 cb($word, i)
函式
reverseText: 回復成原本的 text 內容
