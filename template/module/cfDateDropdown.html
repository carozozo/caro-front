cfDateDropdown
示範
預設
getDate
setDate
disableAll
enableAll
指定下拉選單
程式範例
$('#xx').cfDateDropdown('xxx', {
$year: $dateDropdown.find('.year'),
$month: $dateDropdown.find('.month'),
$day: $dateDropdown.find('.day')
});
指定年份範圍
程式範例
$('#xx').cfDateDropdown('xxx', {
startYear: 1999,
endYear: 2001
});
利用 onSetYear 和 onSetMonth
程式範例
$('#xx').cfDateDropdown('xxx', {
onSetYear: function(i) {
if (i < 2000) {
return false;
}
},
onSetMonth: function(i) {
if (i > 3 && i < 10) {
return true;
}
onSetDay: function(i) {
if (i > 10) {
return false;
}
}
});
簡介
自動化日期下拉選單, 日期下拉選單的值會依照所選的年和月改變
e.g. 閏年的2月, 日期範圍會是 1~29 而不是 1~28
參數
$('xxx').cfDateDropdown(triggerName, opt = {})
triggerName: 指定年和月在 on change 的 namespace, 避免多重觸發
opt 參數
$year: 指定年份 <select> 容器, 未設置則由外掛自動產生
$month: 指定月份 <select> 容器, 未設置則由外掛自動產生
$day: 指定日期
startYear: 起始年份
endYear: 結束年份
onSetYear: 設置每個 year 的 options 時觸發的 cb, return true 會跳過, return false 則終止設置
onSetMonth: 設置每個 month 的 options 時觸發的 cb, return true 會跳過, return false 則終止設置
onSetDay: 設置每個 day 的 options 時觸發的 cb, return true 會跳過, return false 則終止設置
函式
setDate(year, month, day): 設置下拉選單選取的日期
getDate([separator='/']): 取得選擇的日期, 如沒選擇會回傳 null
disableAll(): disable 全部下拉選單
enableAll(): enable 全部下拉選單
