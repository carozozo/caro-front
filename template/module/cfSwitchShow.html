cfSwitchShow
示範
顯示方式
顯示下一個
顯示上一個
position: absolute
display: inline-block
簡介
輪流切換 DOM 內容, 預設顯示第一個, 其他透明度設為 0
參數
$('xxx').cfSwitchShow($domList, [opt = {}])
$domList: DOM 內容組成的陣列
opt 參數
defType: 預設顯示方式, 預設 'fade'
duration: 預設切換時間, 預設 0.5
distance: 預設移動距離,只對 left/right/up/down 有效, 預設 30
befSwitch: 切換之前呼叫的 cb(目前的 index, 要切換的 index), return false 則不切換
aftSwitch: 切換之後呼叫的 cb(目前的 index, 要切換的 index)
函式
switchDom(index, type, opt): 顯示指定的內容
參數
index: $domList 裡的 index
type: 顯示方式
opt 參數
duration: 切換時間, 預設 0.5
distance: 移動距離,只對 left/right/up/down 有效, 預設 30
befSwitch: 切換之前呼叫的 cb(目前的 index, 要切換的 index), return false 則不切換
aftSwitch: 切換之後呼叫的 cb(目前的 index, 要切換的 index)
showNext(type, opt): 顯示下一個內容
參數
type: 顯示方式
opt 參數
duration: 切換時間, 預設 0.5
distance: 移動距離,只對 left/right/up/down 有效, 預設 30
befSwitch: 切換之前呼叫的 cb(目前的 index, 要切換的 index), return false 則不切換
aftSwitch: 切換之後呼叫的 cb(目前的 index, 要切換的 index)
showPrev(type, opt): 顯示上一個內容
參數
type: 顯示方式
opt 參數
duration: 切換時間, 預設 0.5
distance: 移動距離,只對 left/right/up/down 有效, 預設 30
