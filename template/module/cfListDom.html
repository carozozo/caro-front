cfListDom
示範
setList
list反方向
改變寫入值並跳過第二筆
程式範例
list = [{name:'caro1', age:1}, {name:'caro2', age:2}, {name:'caro3', age:3}]
listKey="name" listKey="age"
html
<table border="1" cellpadding="5">
<tr class="listTr">
<td listKey="name"></td>
<td listKey="age"></td>
</tr>
</table>
js
var list = [{name: 'caro1', age: 1}, { name: 'caro2', age: 2}, {name: 'caro3', age: 3}];
$('.listTr').cfListDom(list, {
cb: function(data, i, $dom) {
if (i === 1) {return true;}
$dom.css({color: '#00f'});
data.name = '姓名: ' + data.name;
data.age = '年齡: ' + data.age;
}
});
簡介
將 object-array 資料放到 DOM, 以 list 方式呈現, 例如得獎名單
DOM 本身需要寫入 [listKey='xxx'] 標籤, 用來對應 object 裡面的 key
參數
$('xxx').cfDateDropdown(objArr, opt = {})
objArr: 以 object 組成的 array
opt 參數
befInsert: 每次置入 DOM 之前會呼叫的 cb(object,index,DOM); 回傳 true 會跳過, false 則停止繼續寫入下一筆
befInsert: 每次置入 DOM 之後會呼叫的 cb(object,index,DOM)
isAfter: list DOM 是否是放在 after
